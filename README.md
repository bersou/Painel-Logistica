O **Sistema de Gerenciamento de Logística Cubbo** é uma plataforma avançada (Single-Page Application) desenvolvida para gerenciar, otimizar e rastrear processos logísticos de ponta a ponta. Focado na persona do Gerente de Logística, o sistema permite o controle de estoque em tempo real (WMS), monitoramento de SLAs e o rastreamento preciso de frotas e pacotes.

## ✨ Principais Funcionalidades

- **📊 Dashboard Interativo:** Visão geral da operação com KPIs vitais (Entregas do dia, pacotes em trânsito, ocorrências e taxa de sucesso do SLA).
- **📦 Gestão de Estoque (WMS):** Listagem em tempo real do inventário com indicativos visuais de status (Adequado, Baixo, Crítico) separados por clientes B2B (ex: Magazine Luiza, Kabum).
- **🚚 Rastreamento de Entregas:** Sistema de busca por código (Tracking ID) com retorno em tempo real sobre a localização atual, Previsão de Chegada (ETA) e linha do tempo histórica do envio.
- **📱 Mobile-First & Responsivo:** Interface fluida que se adapta perfeitamente a dispositivos móveis, tablets e desktops. Ideal para equipes operacionais em campo.
- **📲 PWA (Progressive Web App):** O sistema é instalável como um aplicativo nativo em smartphones e desktops, garantindo acessibilidade rápida e imersiva.
- **🔗 Link Preview (Open Graph):** Configuração completa de SEO e metadados para compartilhamento rico em redes sociais e mensageiros (WhatsApp, LinkedIn, etc.).

## 🚀 Tecnologias Utilizadas

Este protótipo foi construído utilizando um ecossistema moderno de desenvolvimento web:

* **[React](https://reactjs.org/)** - Biblioteca JavaScript para construção de interfaces de usuário.
* **[Vite](https://vitejs.dev/)** - Ferramenta de build super rápida e servidor de desenvolvimento.
* **[Tailwind CSS](https://tailwindcss.com/)** - Framework CSS *utility-first* para estilização rápida, responsiva e customizada.
* **[Lucide React](https://lucide.dev/)** - Biblioteca de ícones SVG limpos e modernos.
* **Open Graph / Meta Tags** - Para otimização de compartilhamento e *link previews*.
